 Sports Team Player Information Viewer

A Java Swing-based desktop application developed to manage sports team player records through an intuitive graphical user interface. The application enables users to add, search, display, and delete player information while storing records persistently using CSV files.

1. Features

- Add new player records
- Display all player information
- Search players by name or jersey number
- Delete all player records with confirmation
- Input validation for mandatory fields
- Persistent CSV file storage
- Exception handling for reliable file operations
- Simple and user-friendly graphical interface

2.Technologies Used

- Java
- Java Swing
- Object-Oriented Programming (OOP)
- File Handling
- Collections Framework (ArrayList)
- Event-Driven Programming
- Exception Handling
- CSV File Storage


3. How to Run

- Clone the repository.

```
git clone https://github.com/Srivatsan-2006/Sports-Team-Player-Information-Viewer.git
```

- Open the project in IntelliJ IDEA, Eclipse, or NetBeans.

- Compile and run `PlayerInfoViewer.java`.
